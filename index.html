An exploration in displaying real-time MBTA bus data using a Raspberry Pi.

Madmin

An Administative Console for generating mock services with RESTful URIs.

Outside of our normal mocking and stubbing in unit tests, when building data-heavy, enterprise applications we have found it useful to run a 'live' mock server for proper integration testing for client-side developers as the services are being developed.

See Project
